Combinator example

  1. descendant selector (space) : div p {color:green;} -> Every P inside Div.
  2. child selector (>) : div > p {color:green;} -> Every P inside Div down 1 level no more than that.
  3. adjacent sibling selector (+) : div + p {color:green;} -> Only P next to Div.
  4. general sibling selector (~) : div ~ p {color:green;} -> Every P same level with Div.
